Issues on SecureX will always be entitled under bypass however try to entitle the case under contract/subscription first before bypassing entitlement, some security solutions integrating into SecureX can be used as the entitlement, like AMP for Endpoints, Cisco Umbrella, Email Security (ESA, CES), Web Security (WSA, CWS), Stealthwatch, DUO, Tetration, Meraki and ThreatGRID.
These devices will be entitled to support under bypass if warranty is expired or if there is no contract, we will only deny support only if device is EOL/EOS. An easy way to identify SB devices is by the PID:

Routers: RV
Switches: SG, SF, CBS, SLM, SRW or ESW
Wireless: WAP or CBW
Voice: (this is mostly for phones) 3PCC, MPP (Multiplatform), 3PW, 3PC, 3PD, 3PB, DBS
Not all products might include them, but it is a starting point to properly identify when we need to route to STAC.
Acquisition, they work with a subscription so even if no contract is found on profile or CCW-R we can bypass entitlement for them, before bypassing please make sure to verify if there is a contract or sub. We will deny support ONLY if it is EOL/EOS.

ESA (Email Security Appliance): can be a physical device or a virtual appliance, if the CX provides a SN that has C195, C395, C695, or C695F as part of the PID this means that it is for an ESA.
CES (Cloud E-mail Security) and CWS (Cloud Web Security): cloud-based applications, so they won't have a serial number.
SMA (Security Management Appliance): as the ESA, it has physical devices or virtual appliances, if the CX provides a SN that has M195, M395, M695, or M695F as part of the PID, this will be a SMA.
WSA (Web Security Appliance): As the ESA and SMA, it has physical devices or virtual appliances, if the CX provides a SN that has S195, S395, S695, or S695F as part of the PID, this will be a WSA
Most physical appliances will be x195, x395, x695, x695F (it only changes the letter that precedes the number.)
Virtual appliances wont have serial numbers, but they have a VLN which works as the SN on SCM, though the system might not recognize it on CCW-R or the entitlement tab on CSOne.
